# POWERBI_SALES_DASHBOARD
# E-Commerce Sales Dashboard (Power BI)

## Project Overview
This project is an interactive E-Commerce Sales Dashboard built using Power BI and Superstore dataset.  
The dashboard helps analyze sales performance, profit, quantity, customer segments, and state-wise business insights.

---

## Features
- KPI Cards for:
  - Total Sales
  - Total Profit
  - Total Quantity
  - Profit Margin

- Interactive Visualizations:
  - Sales by Category (Bar Chart)
  - Sales by Segment (Donut Chart)
  - Sales & Profit by State (Map)
  - Profit Contribution by Category (Waterfall Chart)
  - Sales Details Table

- Dynamic Filters (Slicers):
  - Region
  - Category
  - Segment

---

## Tools & Technologies Used
- Power BI
- Data Visualization
- Data Cleaning
- DAX Measures
- Superstore Dataset

---

## Business Insights
- Technology category generated the highest sales.
- Consumer segment contributed the highest share of sales.
- State-wise analysis helped identify high-performing regions.
- Waterfall chart shows category-wise contribution to overall profit.
